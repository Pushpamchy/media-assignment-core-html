Full Stack Web Development Tech Stack

Mean Stack
Mean stands for MongoDB,Express.js, Angular and Node.js, it's a Javascript-based tech stack for building dynamnic web applications
Mern Stack
Mern is similar to mean but uses react instead of Angular for the front-end. It includes MongoDB for Database and Express.js for server
Lamp Stack
Lamp stands for Linux, Apache, MySQL, and PHP/Python/perl. It's a popular open-source stack for building web application.
